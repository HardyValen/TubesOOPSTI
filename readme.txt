TATA CARA MENJALANKAN PERMAINAN PLANT VS ZOMBIE
1. Pull dari branch Milestone1-Windows-v1.0.0 untuk pengguna Windows ATAU branch Milestone1_Linux-v1.0.0 bagi pengguna Linux
2. Buka Command Prompt / Terminal pada folder tempat menyimpan file-file permainan "Plant vs Zombie"
3. Ketik "java Main" (tanpa tanda kutip) untuk menjalankan program permainan
